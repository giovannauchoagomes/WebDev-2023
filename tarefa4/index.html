O que aprendi:

  • Aula 1 - HTML
    Tópicos
    1. Anatomia e Atributos
      • Tags abre e fecha contém a abertura e fechamento e o conteúdo.
      • O conjuto que representa uma tag é um elemento.
      • Cada tag pode ter como atributo uma id, forma que nos refenrenciaremos posteriromente a ela.
      • Podemos concatenar tags.
    2. Tags autocontidas
      • Tag para quebrar linha: <"br/">
      • Tag para inserir imagem <"img height= width= src=/">
    3. Tags abre e fecha
      • Negrito
      • Itálico
      • Listas ordenadas: "ol"; não ordenadas:"ul"; itens:"li"
      • Anchor para hypertextos: "a"
      • Tamanhos de header: h1-h6
      • Parágrafos:"p"
      • Divisória e Seções: "div", "section"
  • Aula 2 - CSS

    1. Introdução a modificação de tags, classes e categorias de itens, exemplos de modificações:
      • color: #ffffff;
      • background-color: white;
      • border: 1px solid black;
      • border-radius: 5px;
      • width: 10vw;
      • display: flex;
      • flex-wrap: wrap;
      • padding: 1em;
      • display: flex;
      • transform: scale(1.5);
      • position: sticky;
      • transition: 0.5s;

    2. Como modificar de animações,pseudo-classes como :hover, :visited, :link e :active.
      Construção: selector:pseudo-class.
